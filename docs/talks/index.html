Asgard: Abstraindo orquestradores de containers

Python Brasil 14, Natal. 19/10/2018

O projeto Asgard é um projeto que tem como objetivo abstrair múltiplos aroquestradores de containers. A ideia é que o time de desenvolvimento não precise sem saber o que está rodando as aplpicações por baixo dos panos. Basta que alguém crie uma Aplicação e clique em “Salvar” e essa aplicação vai ganhar um IP e Porta. Ao mesmo tempo o time de infra-estrutura pode escolher qual orquestrador usar.

O código está no Github: https://github.com/b2w-bit/asgard-api
